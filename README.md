### ⚠️ 使用注意
本项目基于Tampermonkey插件实现，使用前请确保浏览器支持并安装了Tampermonkey插件，新建脚本并复制代码即可

在使用本脚本前，请务必在面板的API配置中填入必要参数，否则脚本将无法调用 AI 接口

注意：为了防止后台监测和人机风险分析，本项目只支持保持浏览器页面在前台运行，切勿最小化，否则功能将失效
