## 🚀 快速开始

### ⚠️ 使用注意
本项目基于Tampermonkey插件实现
在使用本脚本前，请务必在脚本的 `CONFIG` 属性中填入以下必要参数，否则脚本将无法调用 AI 接口：

* **@connect**: 填写你的 API 域名
* **apiKey**: 你的身份验证密钥
* **apiUrl**: 接口请求地址
* **model**: 指定使用的模型 ID
